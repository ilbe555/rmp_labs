CoinFlip – приложение для симуляции подбрасывания монет с ведением статистики.

Основные функции:

* Бросок одной или нескольких монет (1, 2, 3) с анимацией.
* Изменение количества монет через кнопки «+»/«-».
* Отображение результатов каждого броска.
* Сбор статистики: общее количество бросков, орлов, решек и их процентное соотношение.
* Сохранение данных между сессиями и сброс статистики.

Структура экранов:

Главный экран: бросок монет, выбор количества, визуализация результатов.

Экран статистики: просмотр накопленных данных и управление (сброс).

Навигация: переход между экранами осуществляется через элементы интерфейса (кнопки).
